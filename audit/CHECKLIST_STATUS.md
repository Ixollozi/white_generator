# Приоритетные пробелы (3–5 задач)

Из статусов «нет / частично», рекомендуемый порядок:

1. **`year` + `domain` в контексте бренда** — закрывает пункт 3 чеклиста; правка `generators/brand_generator.py` + при необходимости layout.
2. **Отдельная страница `services`** — пункт 4; новый ключ в `template.yaml` всех нужных шаблонов + навигация.
3. **Meta keywords (опционально)** — пункт 6; только если нужно для SEO-политики проекта.
4. **Единое логирование CLI** — пункт 16; одна строка на сайт после генерации в `generate_one_site` или `main.py`.
5. **Шум с эффектом** — пункт 11; подключение сгенерированных css/js в `layout.html` или постобработка классов.

Долгий хвост: изображения, preset-уровни, Keitaro API, деплой, JSON-конфиг.
